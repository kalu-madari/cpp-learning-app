C++ Compiler Not Found

To compile and run C++ code, you need a C++ compiler installed on your system.

Quick Setup (recommended):

  1. Download MSYS2 from msys2.org
  2. Run the installer and follow the prompts
  3. Open MSYS2 UCRT64 terminal and run:
    pacman -S mingw-w64-ucrt-x86_64-gcc
  4. Add C:\msys64\ucrt64\bin to your system PATH
  5. Restart this app

Alternative: MinGW-w64

  1. Download MinGW-w64 from winlibs.com
  2. Extract and add the bin folder to your system PATH
  3. Restart this app
